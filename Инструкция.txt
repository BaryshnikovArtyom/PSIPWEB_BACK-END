2. ИНСТРУКЦИЯ ПО УСТАНОВКЕ НА ХОСТИНГ (ИЛИ ЛОКАЛЬНЫЙ СЕРВЕР)
--------------------------------------------------
ШАГ 1: Перенос файлов
1. Зайдите в панель управления хостингом (или используйте FTP-клиент, например FileZilla).
2. Откройте корневую папку вашего сайта (обычно она называется public_html, htdocs или www).
3. Загрузите все файлы и папки (index.html, папки lr21, lr22 и т.д.) в эту директорию.

ШАГ 2: Настройка базы данных
1. В панели управления хостингом найдите раздел "Базы данных MySQL" (или зайдите в phpMyAdmin).
2. Создайте новую базу данных (если возможно, назовите ее amatto_db).
3. Создайте пользователя базы данных и назначьте ему пароль (сохраните эти данные, они понадобятся на следующем шаге).
4. Зайдите в phpMyAdmin на хостинге, выберите созданную базу данных и перейдите во вкладку "Импорт" (Import).
5. Загрузите файл дампа базы данных (файл с расширением .sql, который должен находиться в корне проекта). Нажмите "Вперед", чтобы таблицы создались.

ШАГ 3: Подключение сайта к базе данных
1. Найдите в файлах сайта скрипт, отвечающий за подключение к БД (обычно это файл с названием db.php, connect.php, config.php или внутри файлов index.php в папках лабораторных).
2. Откройте этот файл для редактирования.
3. Измените настройки подключения на те, что выдал вам хостинг:
   - Хост (Host): обычно 'localhost', но хостинг может указать другой.
   - Имя БД (DB Name): укажите имя БД, созданной на Шаге 2.
   - Пользователь (User): укажите пользователя БД, созданного на Шаге 2.
   - Пароль (Password): укажите пароль от пользователя БД.
4. Сохраните изменения.

ШАГ 4: Проверка работы
1. Откройте адрес вашего сайта в браузере. Должно появиться главное меню с выбором лабораторных работ.
2. Для проверки админки перейдите на соответствующую страницу авторизации и используйте логин "1@gmail.com" и пароль "12345".
